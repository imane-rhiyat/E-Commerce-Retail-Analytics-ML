# SQL Scripts

This folder contains all SQL scripts used throughout the project.

| File | Description |
|------|-------------|
| 01_database_setup.sql | Creates the ecommerce database and transactional tables. |
| 02_load_data.sql | Loads the cleaned datasets into MySQL. |
| 03_analytical_queries.sql | Revenue, churn, RFM metrics and monthly trend analysis. |
| 04_customer_dimension.sql | Creates the customer dimension table and analytical views used by Power BI. |
